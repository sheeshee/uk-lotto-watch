# UK National Lottery API Documentation

> **Disclaimer:** This documentation was generated by Claude (claude-sonnet-4-6) by polling the API endpoints directly and inferring their usage from the responses. It is not official documentation and may be incomplete or inaccurate.

**Base URL:** `https://api-dfe.national-lottery.co.uk`
**Origin:** `https://www.national-lottery.co.uk` (include as `Origin` header)
**Built by:** Allwyn (formerly Camelot) — dev URL: `https://api.dev.allwyn.zenitech.co.uk`
**Tech:** OpenAPI-generated TypeScript client, served via Next.js app on CloudFront

All responses are JSON. Prices are in **pence** (divide by 100 for GBP).

---

## Authentication

Most draw-results and reference-data endpoints are **public** (no auth required).
Account, payment, and game-play endpoints require session cookies (`withCredentials: true`).

### Game Info & Upcoming Draws

#### `GET /draw-game/games/slugs?deviceType={deviceType}`
Returns all active game slugs and their gameIds.

```
GET /draw-game/games/slugs?deviceType=WEB_CLIENT

Response:
{
  "gameSlugs": [
    {"gameSlug": "LOTTO", "activeGameId": 1, "associatedGameIds": [1, 6]},
    {"gameSlug": "EUROMILLIONS", "activeGameId": 33, "associatedGameIds": [33]},
    {"gameSlug": "EUROMILLIONS_HOTPICKS", "activeGameId": 5, "associatedGameIds": [5]},
    {"gameSlug": "LOTTO_HOTPICKS", "activeGameId": 2, "associatedGameIds": [2, 7]},
    {"gameSlug": "SET_FOR_LIFE", "activeGameId": 3, "associatedGameIds": [3]},
    {"gameSlug": "THUNDERBALL", "activeGameId": 4, "associatedGameIds": [4]}
  ]
}
```

**Required:** `deviceType` query param.

#### `GET /draw-game/games/{gameId}?deviceType={deviceType}`
Returns details of the **next upcoming draw** for a game, including jackpot, draw time, sell-close time, ball rules, and game status.

```
GET /draw-game/games/1?deviceType=WEB_CLIENT

Response (Lotto example):
{
  "gameId": 1,
  "topPrize": {"prizeCents": 380000000, "nonCashPrize": null},  // £3.8M jackpot
  "drawDate": "2026-03-07T19:45:00.000Z",                        // Draw time (UTC)
  "isPrizeRollDown": false,
  "sellEndDate": "2026-03-07T19:30:00.000Z",                     // Ticket sales close
  "gameStatus": {
    "status": "OPEN",                                            // OPEN | CLOSED | SUSPENDED
    "endDate": "2026-03-07T19:30:00.000Z"
  },
  "gamePhase": "INITIAL",                                        // null for most games
  "costsPerRowCents": 200,                                       // £2 per line
  "gameRules": {
    "drawnBallsQuantity": {"drawnPrimaryBalls": 6, "drawnSecondaryBalls": 1},
    "primaryRules": {
      "totalQuantity": {"first-ball-number": 1, "last-ball-number": 59},
      "pickBallsQuantitiesOptions": [6],
      "directDebitAvailable": true
    }
  }
}
```

**Required:** `deviceType` query param.

#### `GET /draw-game/games/{gameId}/game-status?deviceType={deviceType}`
Returns just the current game status (lighter than full game endpoint).

```json
{"status": "OPEN", "endDate": "2026-03-07T19:30:00.000Z"}
```

#### `GET /draw-game/games/{gameId}/raffle-events?deviceType={deviceType}&dateFrom={date}&dateTo={date}`
Returns upcoming raffle events (e.g. UK Millionaire Maker for EuroMillions) within a date range.

```
GET /draw-game/games/33/raffle-events?deviceType=WEB_CLIENT&dateFrom=2026-03-01&dateTo=2026-03-31

Response:
{
  "raffleEventDraws": [
    {"raffleType": "UK_MILLION_MAKER", "drawDate": "2026-03-06T20:45:00.000Z", "drawDay": "FRIDAY"},
    {"raffleType": "UK_MILLION_MAKER", "drawDate": "2026-03-10T20:45:00.000Z", "drawDay": "TUESDAY"},
    ...
  ]
}
```

**Required:** `deviceType`, `dateFrom`, `dateTo` (all mandatory).

#### `GET /draw-game/games/next-draw/{playerId}?deviceType={deviceType}`
Returns next draw info personalised to a player (requires auth + playerId).

Returns full prize breakdown for a specific draw number. Same schema as `/latest`.

**Navigation:** Response includes `nextDrawNo` and `previousDrawNo` for traversal.

#### `GET /draw-game/results/{gameId}/draw-dates`
Returns list of draw numbers and dates for the last ~50 draws.

```json
{
  "drawDates": [
    {"drawNo": 3101, "drawDate": "2025-09-10T00:00:00.000Z"},
    {"drawNo": 3102, "drawDate": "2025-09-13T00:00:00.000Z"},

  ]

```

#### `GET /draw-game/results/jackpot-card/{gameId}/{drawNo}`
Returns a summary card for a specific draw's jackpot.

```
GET /draw-game/results/jackpot-card/1/3151

Response:
{
  "topPrize": {"prizeCents": 1221909765, "nonCashPrize": null},
  "amountOfPeople": 33,          // number of Match 5 winners (rolldown context)
  "amountWon": {"prizeCents": 175000, "nonCashPrize": null}
}
```

#### `GET /draw-game/results/{gameId}/download[?interval={N}]`
Downloads draw history as **CSV**. Optional `interval` parameter (number of draws).

```
Content-Type: text/csv

DrawDate,Ball 1,Ball 2,Ball 3,Ball 4,Ball 5,Ball 6,Bonus Ball,Ball Set,Machine,DrawNumber
04-Mar-2026,20,33,43,46,50,56,45,L6,Lotto5,3151
28-Feb-2026,12,15,22,28,31,32,23,L5,Lotto4,3150
...
```

#### `POST /draw-game/results/{gameId}/winning-numbers`
Check if a set of numbers won. Requires request body (`checkWinningNumbersRequest`). Auth may be required.

## Example: Get All Upcoming Draws

```bash
# Get next draw info for all active games
GAMES=(1 2 3 4 5 33)
for gameId in "${GAMES[@]}"; do
  curl "https://api-dfe.national-lottery.co.uk/draw-game/games/$gameId?deviceType=WEB_CLIENT"
done
```

## Example: Get Latest Results

```bash
# All games latest result (single request)
curl "https://api-dfe.national-lottery.co.uk/draw-game/results"

# Specific game with full prize breakdown
curl "https://api-dfe.national-lottery.co.uk/draw-game/results/33/latest"
```

## Notes

- **Prices:** All `prizeCents` values are in **pence** (divide by 100 for £). EuroMillions jackpots represent the UK share only.
- **Draw dates in results** include time (UTC) — the actual draw time. `draw-dates` endpoint returns dates without time.
- **Game 6 / Game 7** are legacy/inactive IDs for Lotto and Lotto HotPicks respectively.
- **EuroMillions (gameId 33)** includes `ukWinnersCount` and `allWinnersCount` separately (multi-country draw).
- **Set For Life (gameId 3)** uses non-cash prizes (`nonCashPrize: "£10,000 every month for 30 years"`).
- The API uses **AWS WAF** (`422a7e6a3d77.edge.sdk.awswaf.com`) and **CloudFront** CDN.
- No rate limiting was encountered during documentation research.
